Удалить тег локально:
```
git tag -d v0.1.0
```
 Удалить тег с GitHub:

```
git push origin :refs/tags/v0.1.0
```
 Проверить список локальных тегов:

```
git tag
```
Создать новый тег
```
git tag v0.1.0
git push origin v0.1.0
```
